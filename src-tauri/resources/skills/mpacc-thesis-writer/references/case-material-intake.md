# Case Material Intake

MPAcc thesis quality depends on case evidence. Treat materials as an evidence inventory before drafting.

## Credibility Levels

| Level | Material | Use |
|---|---|---|
| A | annual reports, prospectus, bond documents, regulatory inquiries/replies, accounting standards, official policies, statistical yearbooks | directly cite with page/date/source |
| B | industry reports, brokerage research, reputable news, court/regulatory announcements | cite after cross-checking |
| C | interview notes, questionnaires, internship/internal materials | use with desensitization and explain acquisition method |
| D | unsourced web posts, memory, informal statements | do not use as evidence without verification |

## Intake Fields

`材料名称 / 类型 / 时间范围 / 来源链接或页码 / 关键指标 / 可信度 / 是否可公开引用 / 对应章节 / 缺口说明`

Use `assets/case-intake-template.md`.

Also tag each important claim as:

- directly verifiable: supported by annual reports, announcements, inquiry replies, bond documents, regulations, court/regulatory documents, or supplied case files;
- reasonable inference: supported by multiple public materials but still inferential;
- unverifiable limit: depends on internal decisions, contracts, ledgers, credit files, supplier terms, or meeting minutes not available to the writer.

Do not write unverifiable limits as factual conclusions.

## Data Gap Strategy

1. Build a verifiable skeleton from public data.
2. Mark gaps with placeholders instead of inventing facts.
3. Ask for the minimum extra materials needed to support each claim.
4. Prefer standard indicators and public events for the core problem; use custom calculations only as supplementary evidence with transparent formulas.

Placeholder examples:

- `〔待补：A 公司 2021-2023 年预算执行偏差率，来源建议：预算执行表或财务部门访谈〕`
- `〔待补：同行业 2-3 家公司环境成本披露口径，用于横向比较〕`

Good: "This conclusion is provisional because interview evidence from the finance department is missing."

Weak: "Assume the company has weak internal control and continue drafting."
